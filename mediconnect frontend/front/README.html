Connexion backend Spring Boot

Ce frontend HTML/CSS/JS est conçu pour être connecté au backend Spring Boot (port 8080). Chaque page contient les endpoints commentés :

  • POST /api/auth/login → Authentification JWT
  • GET /api/appointments/me → Rendez-vous patient
  • GET /api/doctors → Recherche médecins
  • GET /api/admin/stats → Statistiques admin
  • WS /ws → Chat temps réel STOMP/SockJS

Pour la version Angular 17 complète avec HttpClient, intercepteurs JWT et WebSocket → voir le projet mediconnect-angular.zip
